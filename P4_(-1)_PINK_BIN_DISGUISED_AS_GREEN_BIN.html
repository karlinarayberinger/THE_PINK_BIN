PINK_BIN_DISGUISED_AS_GREEN_BIN

Each of the following web page links below leads to a particular directory page which displays a list of component web pages. Most of those component web pages displays a single screenshot image taken of social media posts made by Karlina Ray Beringer (@karbytes). Each of the screenshot images was backed up to Archive dot Org using the WayBack Machine.

The directory pages whose index numbers are 53 to 56 were the author’s attempts to get out of the pink bin and into the green bin as quickly as possible only to later discover that the pink was an insidious poison which seeped into those directories and revealed them to be pink bin directories disguised as green bin directories. It’s akin to a person being asleep and thinking that it woke up from a dream only to find that it is still asleep inside of some dream. (Yes, it is theoretically possible to have dreams inside of dreams).
